# CodeAlpha_C-Programming
# C Programming Projects

A collection of four C programming projects demonstrating fundamental to advanced programming concepts. These projects cover arithmetic operations, matrix manipulation, file handling, structures, and menu-driven programming.

---

## 📌 Project Overview

This repository contains the following tasks:

| Task | Project | Difficulty |
|------|---------|------------|
| Task 1 | Calculator Program | Basic |
| Task 2 | Matrix Operations | Medium |
| Task 3 | Student Management System | High |
| Task 4 | Bank Account Management System | High |

# 📌 Task 1: Calculator Program

## Objective

Develop a simple calculator that performs basic arithmetic operations using the **switch-case** statement.

### Features

- Addition
- Subtraction
- Multiplication
- Division
- Division by zero handling
- User-friendly menu

### Concepts Used

- Variables
- Input/Output
- Switch Case
- Conditional Statements
- Arithmetic Operators

### Sample Output

```
Select Operation

1. Addition
2. Subtraction
3. Multiplication
4. Division

Enter Choice: 1

Result = 25
```
